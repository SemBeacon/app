## About
This repository contains the SemBeacon Android and iOS application.

## Privacy Policy
More information about what data the SemBeacon application collects and how this information is used can be read in the online privacy policy: https://sembeacon.org/privacy-policy/

## Contributing
Contributions to the SemBeacon application are welcome. Any contributions to the specifications should be made in the [specification](https://github.com/SemBeacon/specification) repository while any contributions to the application should be made to this repository.
